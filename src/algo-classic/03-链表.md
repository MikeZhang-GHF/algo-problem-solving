[Q206] Reverse Linked List
> - 反转链表，迭代和递归两种方法。
> - 时间复杂度: $`O(n)`$，其中n为链表长度。
> - 空间复杂度: $`O(1)`$。

- 迭代写法
```python
class Solution:
    def reverseList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        pre, cur = None, head
        while cur:
            nxt = cur.next
            cur.next = pre 
            pre = cur
            cur = nxt
        return pre 
```
