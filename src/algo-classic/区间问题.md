[Qzj-future01] 信号接收
- 区间重叠问题，左端点排序，上一个区间的右端点和下一个区间的左端点是否重合，这里是左闭右开区间，所以是`<=`。
- 时间复杂度$`O(nlogn)`$
- 空间复杂度$`O(n)`$。

```python
class Solution:
    def canReceiveAllSignals(self, intervals: List[List[int]]) -> bool:
        intervals.sort()
        return all(r <= l for (_, r), (l, _) in pairwise(intervals))
```
