[Q2680] 最大或值

```python
class Solution:
    def maximumOr(self, nums: List[int], k: int) -> int:
        n = len(nums)
        suf = [0] * (n + 1)
        for i in range(n-1, 0, -1):
            suf[i] = suf[i+1] | nums[i]
        
        ans = pre = 0
        for i, x in enumerate(nums):
            ans = max(ans, pre | (x << k) | suf[i+1])
            pre |= x 
        return ans 
```
