### [Q6424] -  最小化字符串长度 - Easy
#### Solution:

- High Level:
  > 就是统计字符串中的不同字符的个数。

- Algorithm Knowledge:
  > 使用一个整数表示一个字符串的不同字符的个数。

- Code Tricks:
  > 统计一个二进制数中1的个数。


#### Code:

- Python
```python
class Solution:
    def minimizedStringLength(self, s: str) -> int:
        mask = 0
        for c in s:
            mask |= 1 << (ord(c) - ord('a'))
        return bin(mask).count('1')
```
- Java
```java
class Solution {
    public int minimizedStringLength(String s) {
        int mask = 0;
        for (char c: s.toCharArray())
            mask |= 1 << (c - 'a');
        return Integer.bitCount(mask);
    }
}
```

- Typescript
```typescript
function minimizedStringLength(s: string): number {
    let mask = 0;
    for (let c of s) {
        mask |= 1 << (c.charCodeAt(0) - 'a'.charCodeAt(0));
    }
    // mask.toString(2)变成二进制, 然后按照1分割
    return mask.toString(2).split('1').length - 1;
}
```
