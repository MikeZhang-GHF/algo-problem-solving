### [Q1] - 200. 岛屿数量
#### Solution:

- High Level:
  > 

- Algorithm Knowledge:
  > BFS

- Code Tricks:
  > BFS template
  > 1. two queue
  > 2. one queue
  > grid expand to 4 directions 


#### Code:
```python
class Solution:
    def numIslands(self, grid: List[List[str]]) -> int:
        m, n = len(grid), len(grid[0])
        vis = set()

        def bfs(i, j):
            vis.add((i, j))
            q = [(i, j)]
            while q:
                tmp = q
                q = []
                for i, j in tmp:
                    # four directions and is land, expand
                    for x, y in (i+1, j), (i-1, j), (i, j+1), (i, j-1):
                        if 0 <= x < m and 0 <= y < n and grid[x][y] == '1' and (x,y) not in vis:
                            q.append((x,y))
                            vis.add((x,y))
                
        ans = 0
        for i in range(m):
            for j in range(n):
                if grid[i][j] == '1' and (i,j) not in vis:
                    bfs(i, j)
                    ans += 1
        return ans
```
