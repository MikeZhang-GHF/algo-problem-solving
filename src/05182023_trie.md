### [Q2414] - 最长的字母序连续子字符串的长度
#### Solution:

- High Level:
  > 本质寻找某个特性的分组模型。比如寻找一个字符串中的第一个数字，可以使用regex。可以使用双指针。
  > 找到起点，然后找到终点，维护某种信息，如最大值。

- Algorithm Knowledge:
  > 双指针

- Code Tricks:
  > 别忘记处理最后一个分组


#### Code:
```python
class Solution:
    def longestContinuousSubstring(self, s: str) -> int:
        # 分组
        ans = start = 0
        n = len(s)
        for i in range(1, n):
            if ord(s[i]) != ord(s[i-1]) + 1:
                # 找到了一个组的末尾
                ans = max(ans, i - start)
                start = i 
        return max(ans, n - start)
```
