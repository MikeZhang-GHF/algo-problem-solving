### [Q2414] - 最长的字母序连续子字符串的长度 - Medium
#### Solution:

- High Level:
  > 本质寻找某个特性的分组模型。比如寻找一个字符串中的第一个数字，可以使用regex。可以使用双指针。
  > 找到起点，然后找到终点，维护某种信息，如最大值。

- Algorithm Knowledge:
  > 双指针

- Code Tricks:
  > 别忘记处理最后一个分组


#### Code:
- Python

```python
class Solution:
    def longestContinuousSubstring(self, s: str) -> int:
        # 分组
        ans = start = 0
        n = len(s)
        for i in range(1, n):
            if ord(s[i]) != ord(s[i-1]) + 1:
                # 找到了一个组的末尾
                ans = max(ans, i - start)
                start = i 
        return max(ans, n - start)
```

### [Q2415] -  反转二叉树的奇数层 - Medium


#### Solution:

- High Level:
  > 关键字：层, 反转 -> BFS

- Algorithm Knowledge:
  1. BFS
  > BFS 在树上的应用
  > BFS 模板，
  > 1.所有起点都放入队列, 维护一个标记访问过的节点的集合
  > 2. 队列不空，进行状态转移。

  2. DFS
  > DFS 在树上的应用
  >
  >

- Code Tricks:
  > 不知道上一个节点，知道下一个节点。
  > 如果直接换节点，非常麻烦，可以换值。
  > x ^= 1, 0 -> 1, 1 -> 0, 奇偶变换。

#### Code:
- Solution 1: BFS
- Python

```python
class Solution:
    def reverseOddLevels(self, root: Optional[TreeNode]) -> Optional[TreeNode]:
        q = [root]
        level = 1
        while q[0].left:
            nxt = []
            for node in q:
                nxt += [node.left, node.right]
            q = nxt
            # 奇数层换值，不换节点，偷天换日
            if level:
                for i in range(len(q)//2):
                    x, y = q[i], q[len(q) - i - 1]
                    x.val, y.val = y.val, x.val 
            level ^= 1  # 奇数偶数变化
        return root 
```

- Solution 2: DFS
- Python

```python
class Solution:
    def reverseOddLevels(self, root: Optional[TreeNode]) -> Optional[TreeNode]:
        def dfs(node1, node2, is_odd_level):
            if node1 is None:
                return 
            if is_odd_level:
                # 交换节点值
                node1.val, node2.val = node2.val, node1.val 
            dfs(node1.left, node2.right, not is_odd_level)
            dfs(node1.right, node2.left, not is_odd_level)

        dfs(root.left, root.right, True)
        return root 
```

```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def isSymmetric(self, root: Optional[TreeNode]) -> bool:
        def dfs(left, right):
            if left is None or right is None:
                return left is None and right is None  
            if left.val != right.val:
                return False 
            return dfs(left.right, right.left) and dfs(left.left, right.right)
        return dfs(root, root)
```

### [Q2416] -  字符串的前缀分数和 - Hard

#### Solution:

- High Level:
  > 关键字 前缀，前缀的个数 -> Trie
  > 1. 前缀的个数，可以使用Trie来统计。
  > 2. 在Trie上，进行DFS，一直走到叶子节点，统计路径上的前缀的个数，就是word的分数。

- Algorithm Knowledge:
  > Trie字典树 + Trie上的DFS

- Code Tricks:
  > Trie字典树模板
  > 对于python,使用 __slots__ 进行加速


#### Code:

#### Python 
```python
class Node:
    # 加速对象创建
    __slots__ = 'son', 'score', 'ids'

    def __init__(self):
        self.son = defaultdict(Node)
        self.score = 0
        self.ids = []

class Solution:
    def sumPrefixScores(self, words: List[str]) -> List[int]:
        root = Node()
        for i, word in enumerate(words):
            cur = root 
            for c in word:
                cur = cur.son[c]
                cur.score += 1 # 维护经过该节点字符串的个数，前缀
            cur.ids.append(i) # 记录末尾，并且记录单词的下标，为了答案，可能有重复字符串
            
            # 在trie上dfs，走到字符串末尾时的路径分数之和
        ans = [0] * len(words)
        def dfs(node: Node, sum: int) -> None:
            sum += node.score  # 累加分数，即可得到答案
            # 更新这个节点为结尾的分数
            for i in node.ids:
                ans[i] = sum
            for child in node.son.values():
                if child:
                    dfs(child, sum)
        dfs(root, 0)
        return ans 
```

#### Java

```java
class Node {
    Map<Character, Node> son;
    int score;
    List<Integer> ids;

    public Node() {
        son = new HashMap<>();
        score = 0;
        ids = new ArrayList<>();
    }
}


class Solution {
    public int[] sumPrefixScores(String[] words) {
        int n = words.length;
        Node root = new Node();

        for (int i = 0; i < n; i++) {
            String word = words[i];
            Node cur = root;
            for (char c: word.toCharArray()) {
                cur = cur.son.computeIfAbsent(c, k -> new Node());
                cur.score++;
            }
            cur.ids.add(i);
        }

        int[] ans = new int[n];
        dfs(root, 0, ans);
        return ans;
    }

    private void dfs(Node node, int sum, int[] ans) {
        if (node == null) return;
        sum += node.score;
        for(int idx: node.ids)
            ans[idx] = sum;
        for(Node child: node.son.values())
            if (child != null)  
                dfs(child, sum, ans);
    }
}
```

[Q208] - 实现 Trie (前缀树) - Medium
> Trie模板题
> 字典树的性质
> 1. trie可以看作多叉树，如果使用哈希表实现，如果使用数组实现，可以看作26叉树。
> 2. trie的每个节点表示一个字符串的前缀，根节点表示空字符串。节省空间。比如统计经过某个节点x(前缀)的字符串的个数，可以在x上维护一个cnt。
  
```python
class Node:
    __slots__ = 'son', 'is_word'

    def __init__(self):
        # 儿子节点都是字典，不用数组26来实现
        self.son = defaultdict(Node)
        self.is_word = False

class Trie:

    def __init__(self):
        self.root = Node()

    def insert(self, word: str) -> None:
        cur = self.root 
        for c in word:
            cur = cur.son[c]
        cur.is_word = True 

    def search(self, word: str) -> bool:
        cur = self.root 
        for c in word:
            if c not in cur.son:
                return False 
            cur = cur.son[c]
        return cur.is_word

    def startsWith(self, prefix: str) -> bool:
        cur = self.root
        for c in prefix:
            if c not in cur.son:
                return False 
            cur = cur.son[c]
        return True 
```
